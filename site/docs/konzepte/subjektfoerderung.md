# Subjektförderung

Finanzielle Unterstützung an Haushalte im Rahmen der Wohnraumförderung (vs. [Objektförderung](/wiki/konzepte/objektfoerderung)).

## Instrumente

Wohngeld, Kosten der Unterkunft (KdU), Heizzuschüsse, Baukindergeld (KfW, 2018-2020).

## Problem

Kann Preise treiben. Lindert Symptome, löst nicht das Angebotsproblem.
