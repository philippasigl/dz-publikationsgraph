# Langzeitarbeitslosigkeit

Arbeitslosigkeit, die länger als ein Jahr andauert — strukturelles Phänomen, das von friktioneller, kurzfristiger Arbeitslosigkeit abzugrenzen ist.

## Funktion

Während friktionelle Kurzzeitarbeitslosigkeit in jeder Marktwirtschaft auftritt, gilt unter [Vollauslastung](/wiki/konzepte/vollbeschaeftigung) des Arbeitsmarkts, dass niemand dauerhaft erfolglos sucht. DZ nutzt die Abwesenheit von Langzeitarbeitslosigkeit deshalb als operatives Kriterium für Vollbeschäftigung — und damit als Input zur Berechnung des [Produktionspotenzials](/wiki/konzepte/produktionspotenzial) anstelle der statistisch fortgeschriebenen [NAWRU](/wiki/konzepte/nawru). Auf SGP-Ebene wird die Arbeitslosenquote analog um den Anteil der Langzeitarbeitslosen korrigiert.

## DZ-Perspektive

DZ schlägt vor, dass das Arbeitspotenzial als ausgeschöpft gilt, sobald keine Langzeitarbeitslosigkeit mehr vorherrscht — eines von drei Kriterien des Reformvorschlags für die [Konjunkturkomponente](/wiki/konzepte/konjunkturkomponente) der Schuldenbremse. Dieser Ansatz löst die NAWRU-Probleme (Pro-Zyklik, fehlende Robustheit) und macht Vollbeschäftigung politisch operationalisierbar.
